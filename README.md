# Mira Vista Site

Starter website for `mira-vista.com`.

## Cloudflare Pages settings

Use these settings when connecting the GitHub repository:

```text
Framework preset: None
Build command: leave empty
Build output directory: /
```

After deployment, add these custom domains in the Pages project:

```text
mira-vista.com
www.mira-vista.com
```

Cloudflare should create the DNS records automatically.
