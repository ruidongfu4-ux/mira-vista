隐私政策

最后更新：2026 年 5 月 31 日 · English · 服务条款

本隐私政策说明 日省 App（"我们"、"本应用"）如何处理你的信息。日省按 local-first 哲学 设计：你的自省内容永远只存在你的设备上，除了下面明确列出的窄通道之外，不进入任何服务器。

1. 一句话说清楚

内容存储位置谁能读
你对反思问题的回答仅你的设备（SQLite）只有你
你的人格档案、关键词、洞察历史仅你的设备只有你
通知时间、语言偏好仅你的设备只有你
日省给你出的问题、档案摘要、关键词、近期回答实时发送给我们的 AI 服务商（DeepSeek）以生成回应，按其政策处理后丢弃日省（仅传输不存储）+ DeepSeek（按其政策）
你主动长按某个问题选「推荐到社区」时上传的问题文本上传到我们的社区库，不关联你的身份其他日省用户（经我们审核后）

我们 不会：

  • 运行任何追踪 SDK
  • 把你的数据卖给广告商或第三方
  • 要求账号、邮箱、手机号或任何身份标识
  • 把你的自省记录存到任何服务器

2. 我们处理的信息

2.1 仅存于你设备本地的

以下所有数据都仅存在你设备的 iOS 沙箱 SQLite 数据库中，不上传到任何服务器：

  • 你对自省题的回答
  • 从你回答中生成的人格档案（Markdown）
  • AI 从你文字中提取的关键词
  • 你的自省历史（sessions）
  • 你的精选问题与实验
  • 周/月总结
  • 通知设置、App 语言偏好

iOS 文件系统默认对 App 数据加密。

2.2 实时发送给 AI 服务商（DeepSeek）的

当你点击需要 AI 处理的按钮（生成问题、分析回答、写总结、设计实验等）时，日省的后端（一个 Supabase Edge Function）会把必要的上下文发送给 DeepSeek 的 API 来计算结果。

发送内容包括（视操作而定）：

  • 当前的反思问题
  • 你写下的回答
  • 你人格档案的截取片段（≤ 3000 字符）
  • 你的高权重关键词（含定义）
  • 近期 5 天 sessions 的摘要（已截断）

DeepSeek 的政策声明 API 内容不用于训练、不超出请求生命周期存储。我们无法绝对保证任何第三方在其政策之外的行为。

我们 不在自己的服务器上存储 这些内容。日省后端只是设备和 DeepSeek 之间的无状态中转层。

2.3 主动推荐到社区库的内容（仅当你显式选择时）

当你长按一个问题、点「推荐到社区」时，以下内容会上传到我们的社区数据库：

  • 问题文本本身（AI 生成的问题，不是你的回答）
  • 问题类型
  • AI 生成它的理由（why_ask，内部元数据）
  • 时间戳

我们 不会上传：

  • 你的回答
  • 你的档案或关键词
  • 你的设备 ID 或任何身份标识
  • 任何把这条问题与你本人关联起来的信息

被推荐的问题进入待审核池。经过基础自动审核后（我们会拒绝含个人姓名、具体日期、金额、住址的问题），它们可能出现在其他用户的题目列表里，带有 "✦ Community" 标签。

你可以随时停止推荐新的问题。已经匿名提交的问题因系统的匿名性无法由用户单方撤回，但你可以发邮件告诉我们具体文本，我们会从社区池中删除。

3. 我们的服务器

我们使用 Supabase（部署在 AWS 上）运行：

  • 转发请求到 DeepSeek 的 Edge Function
  • 社区问题表（仅存按 §2.3 主动推荐的问题）

Supabase 没有任何你的自省内容、档案、或使用模式的记录。社区表只包含 §2.3 描述的数据。

我们使用 DeepSeek 作为 AI 服务商。它们实时处理你的消息（按其政策不存储）。

未来版本可能切换或新增 AI 服务商（如 OpenAI、Anthropic）。如发生变更，本政策会同步更新。

4. 儿童隐私

日省不面向 13 岁以下用户。我们不会有意收集儿童数据。如你认为有儿童在使用本应用，请联系我们协助。

5. 你的权利

由于你的所有自省记录都仅存在你的设备上：

  • 删除全部数据：用设置 → 数据 → "重新开始 onboarding" 按钮，会清空设备上的所有数据。
  • 删除 App：立即且永久地清除所有数据。我们不保留任何副本。
  • 删除已推荐到社区的某条问题：发邮件告诉我们问题文本，我们会从池中移除。
  • 数据导出：你可以在档案页把人格档案以 Markdown 形式分享出去。

我们的服务器上没有你的数据，所以传统的"请求查看数据"、"请求删除数据"对你的自省记录不适用 — 它们一直就是你的，在你的设备上，没有别处。

6. 政策变更

如果数据处理方式有重大变更，我们会：

  • 更新顶部日期
  • 在 App 内下次启动时通知你
  • （涉及隐私破坏性变更时）请求你重新同意

7. 免责声明

日省是一个 自省与自我观察工具。它不是医疗器械、心理治疗 App、或专业心理咨询的替代品。AI 生成的洞察是自省的工具，不是诊断或治疗建议。

如果你正在经历心理危机，请联系专业人员或心理援助热线。
